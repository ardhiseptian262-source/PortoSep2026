# 🚀 Professional Portfolio Website

A modern, responsive portfolio website built with React.js, Tailwind CSS, and Vite. Showcasing tech skills, projects, and personal achievements with a professional design.

## 📋 Features

- ✅ **Modern Design** - Clean, dark-mode oriented interface with glassmorphism effects
- ✅ **Fully Responsive** - Mobile-first design that works seamlessly from mobile to desktop
- ✅ **Interactive Components** - Smooth animations and transitions for engaging UX
- ✅ **Component-Based Architecture** - Modular, reusable React components
- ✅ **Contact Form** - Fully functional contact form with state management
- ✅ **Skills Showcase** - Display both technical and soft skills with progress levels
- ✅ **Project Gallery** - Beautiful project cards with technology tags and links
- ✅ **Experience Timeline** - Professional achievements and work experience
- ✅ **Built with Modern Stack** - React 18+, Tailwind CSS, TypeScript, Vite

## 🛠️ Tech Stack

- **Framework**: React 18+
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide-React
- **Language**: TypeScript
- **CSS Processing**: PostCSS & Autoprefixer

## 📦 Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Steps

1. **Clone the repository** (if you have it in git)
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
   
   Or with yarn:
   ```bash
   yarn install
   ```

3. **Install additional packages** (if not already in package.json)
   ```bash
   npm install react react-dom
   npm install -D @vitejs/plugin-react vite
   npm install tailwindcss postcss autoprefixer
   npm install lucide-react
   ```

## 🚀 Development

Start the development server:

```bash
npm run dev
```

The application will open in your default browser at `http://localhost:3000`

### Development Features
- Hot module replacement (HMR)
- Fast refresh for React components
- Instant CSS updates

## 🏗️ Building for Production

Create an optimized production build:

```bash
npm run build
```

The build output will be in the `dist` directory, ready for deployment.

### Preview Production Build

To preview the production build locally:

```bash
npm run preview
```

## 🎨 Customization

### Update Content
Edit `src/data/content.ts` to update:
- Technical and soft skills
- Project information
- Work experience
- Achievements

### Modify Colors
Edit `tailwind.config.js` to customize:
- Primary colors (primary, secondary, accent)
- Dark mode settings
- Animation keyframes

### Change Personal Information
Update these sections in the components:
- Hero section headline in `src/components/Hero.tsx`
- Contact information in `src/components/Contact.tsx`
- Social media links in `src/components/Navbar.tsx` and `Footer.tsx`

## 🌐 Deployment

### Deploy to Vercel (Recommended)
1. Push your code to GitHub
2. Connect your repository to Vercel
3. Vercel will auto-detect Vite and deploy automatically

### Deploy to Netlify
1. Build the project: `npm run build`
2. Connect your Git repository to Netlify
3. Set build command to: `npm run build`
4. Set publish directory to: `dist`

### Deploy to Other Platforms
The `dist` folder generated by `npm run build` is production-ready and can be deployed to:
- AWS S3 + CloudFront
- GitHub Pages
- Firebase Hosting
- Any static hosting service
